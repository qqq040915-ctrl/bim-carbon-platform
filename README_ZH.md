# BIM A1-A3 碳排計算分析平台

## 📋 項目概述

這是一個專業的 BIM（建築信息模型）A1-A3 碳排計算分析平台，用於評估建築生命週期的碳足跡。支援從 Revit 匯出的 CSV 或 XLSX 明細表的自動化分析。

---

## ✨ 核心功能

### 1. 碳排係數設定區塊
- **預設係數**：320 kgCO2e/m³（混凝土）
- **可修改**：使用者可依實際查詢資料自行修改
- **即時生效**：修改後系統立即重新計算所有結果
- **資料來源**：環境部碳足跡資訊網

### 2. 文件上傳與識別
- **支援格式**：CSV、XLSX、XLS
- **自動欄位識別**：
  - 體積欄位：支援「體積」、「Volume」、「體積(m³)」、「體積(m3)」
  - 構件類型：支援「族類型」、「族」、「類型」、「family」等
- **構件類型自動分類**：
  - 柱（column）
  - 梁（beam）
  - 樓板（slab/floor）
  - 牆（wall）

### 3. 自動計算
計算公式：**A1-A3 碳排放量 = 構件體積 × 本研究採用碳排係數**

### 4. 多維度結果展示
#### 各構件結果卡片
- 柱：總體積與碳排
- 梁：總體積與碳排
- 樓板：總體積與碳排
- 牆：總體積與碳排

#### 總計摘要
- 總體積
- 總碳排

### 5. 數據可視化
- **圓餅圖**：各構件碳排占比
- **雙軸長條圖**：構件體積與碳排比較

### 6. 詳細數據表
完整的明細表，包括：
- 序號
- 族類型
- 構件類型
- 體積（m³）
- 碳排（kgCO2e）

## 🚀 快速開始

### 安裝依賴
```bash
npm install
```

### 啟動開發伺服器
```bash
npm run dev
```

開發伺服器將在 `http://localhost:5173/` 運行（或其他可用埠）

### 使用步驟

1. **修改碳排係數（可選）**
   - 在「本研究採用碳排係數」區塊中修改數值
   - 預設值為 320 kgCO2e/m³

2. **上傳 Revit 明細表**
   - 點擊「選擇檔案」按鈕
   - 選擇 CSV 或 XLSX 文件
   - 確保文件包含體積欄位和構件類型欄位

3. **查看分析結果**
   - 自動計算各構件碳排
   - 查看總計摘要
   - 檢查圖表可視化
   - 下載或打印詳細數據表

## 📊 數據格式要求

### CSV/XLSX 文件結構
必須包含以下欄位（列名稱支援多種變異）：

| 欄位 | 支援的名稱 | 示例 |
|------|----------|------|
| 構件族類型 | Family, 族類型, 族, Type | column_01, beam_01 |
| 體積 | Volume, 體積, 體積(m³), 體積(m3) | 12.5, 45.6 |

### 數據示例

```csv
Family,Volume
column_01,12.5
column_02,15.3
beam_01,45.6
slab_01,280.5
wall_01,95.4
```

## 📝 注意事項

1. **編碼問題**：確保 CSV 文件使用 UTF-8 編碼
2. **列名稱**：系統會自動識別列名稱，但建議使用標準名稱
3. **數據精度**：計算結果保留 2 位小數
4. **邊界情況**：若未找到體積欄位，系統將顯示警告
